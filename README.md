## Prerequisites

- **Node.js**: v20.x
- **Yarn**: v1.x
- **Docker**: latest version
- **Docker Compose**: latest version

## Running the Application

### Frontend
cd frontend
yarn dev

### Backend
cd backend
yarn start:dev

## Docker Setup

### Running with Docker Compose
docker-compose up

### Stopping the Containers
docker-compose down
